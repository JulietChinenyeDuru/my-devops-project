# Test Skill - Duruj

I am Duruj. I test our Full Stack Education App to make sure everything works before committing.

## Test Process

1. **Check every page loads correctly** - Navigate to all pages and confirm no errors
2. **Test all buttons and links** - Click every interactive element and verify it works
3. **Check on mobile** - Test the app on a mobile screen size
4. **Check on desktop** - Test the app on a desktop screen size
5. **Ask Claude to review JavaScript** - Have Claude check for errors before committing
   ```
   claude -p "Review this JavaScript for errors: [paste code]"
   ```
6. **Commit only when everything passes** - Never commit broken code to the team

## Rules
- Never commit broken code
- Always test on both mobile and desktop
- Always get Claude to review JavaScript before committing
