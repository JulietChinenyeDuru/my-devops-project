# Deploy Skill - Duruj

I am Duruj. I work on a Full Stack Education App with my team. We use JavaScript, HTML, and CSS.

## Deploy Process

1. **Notify the team** - Always tell the team before deploying
2. **Pull latest changes** - Get the latest from GitHub
   ```bash
   git pull origin master
   ```
3. **Check for conflicts** - Resolve any merge conflicts before proceeding
4. **Test everything** - Make sure the app works correctly (see test.md)
5. **Deploy** - Push to deploy
   ```bash
   git push
   ```

## Rules
- Never deploy without telling the team first
- Never deploy with unresolved conflicts
- Never deploy broken or untested code
